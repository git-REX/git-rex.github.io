Autonomous Object Detection and Transport Robot

  • University Project

This project involved building an autonomous robot capable of detecting objects and colours, then grabbing, sorting, and placing them at specific locations. We used LEGO for the chassis, MATLAB for programming, and integrated various sensors along with NI myDAQ for real-time data acquisition. The result was a fully functional prototype showcasing intelligent automation, precise control and efficient system integration.
